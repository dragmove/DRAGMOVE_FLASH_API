public static function getDifferenceBetweenDate(_baseYear:int, _baseMonth:int, _baseDay:int, _baseHour:int, _baseMin:int, _baseSec:int, _targetYear:int, _targetMonth:int, _targetDay:int, _targetHour:int, _targetMin:int, _targetSec:int):Array

base Date로부터 target Date까지의 날짜 및 시간의 차이를 DAY, HOUR, MINUTE, SECOND 정보 순서대로 Array에 담아 반환합니다.

Parameters

_baseYear:int — 현재 연도 - 4자리 연도
 
_baseMonth:int — 현재 월 - 1 ~ 12
 
_baseDay:int — 현재 일 - 1 ~ 31
 
_baseHour:int — 현재 시간 - 0 ~ 23
 
_baseMin:int — 현재 분 - 0 ~ 59
 
_baseSec:int — 현재 초 - 0 ~ 59
 
_targetYear:int — target 연도 - 4자리 연도
 
_targetMonth:int — target 월 - 1 ~ 12
 
_targetDay:int — target 일 - 1 ~ 31
 
_targetHour:int — target 시간 - 0 ~ 23
 
_targetMin:int — target 분 - 0 ~ 59
 
_targetSec:int — target 초 - 0 ~ 59

Returns
Array — Array

Example
Basic usage:
        trace(SystemUtil.getDifferenceBetweenDate(2009, 12, 25, 12, 0, 0, 2010, 12, 25, 12, 30, 30)); //2009.12.25 12시 0분 0초와 2010.12.25 12시 30분 30초와의 차이값인 365,0,30,30 반환 (365일 0시간 30분 30초)
